# Simple password generator

Generate a simple random password with [ASCII](https://www.asciitable.com) encoding.

## Compilation

```
make
```

## Usage

```
make [MODE] [NUMBER]
```

| Mode | Description |
| ---- | ----------- |
| `-n` | Normal mode. Don't accept space. |
| `-s` | Space mode. Accept space. |

| Number | Description |
| ------ | ----------- |
| 0 - 99 | Number of character. |
